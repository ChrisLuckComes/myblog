# Hello React

我的 React 学习笔记。

## 安装

需要 node.js>=8.10 && npm>=5.6

```
npx create-react-app hello_react --typescript
cd hello_react
npm start

```

# 核心概念

## 1. JSX(JavaScript XML)

```JSX
const element=<h1>Hello,world</h1>
```

这就是 JSX，一个js的语法扩展，它可以生成React元素。
